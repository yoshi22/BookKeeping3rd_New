## 🐛 バグ管理・報告

### バグ分類

- **Critical**: アプリクラッシュ、データロス
- **High**: 主要機能不動作
- **Medium**: UI問題、軽微な機能不具合
- **Low**: 文言、デザイン改善

### 報告フォーマット

```markdown

**対象環境**: iOS/Android, デバイス名, OS Version
**再現手順**:

2. ステップ2
   **期待結果**:
```
